A simple command-line based Bus Ticket Booking System written in Python.
This program allows users to book bus seats interactively until all seats are filled.
📌 Features

1 . Displays available seats

2 . Allows users to book seats one by one

3 . Stops booking when seats are full

4 . Handles invalid input

5 . Simple and beginner-friendly logic

🛠️ Technologies Used

Python 3

Basic concepts:

  Variables

  While loop

  Conditional statements (if-elif-else)

  User input handling

📚 Learning Purpose

This project is great for beginners to practice:

  Loop control

  Conditional statements

  User interaction

  Basic program flow
